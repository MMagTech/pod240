Place the AtomicParsley binary here for your platform:

  Windows: AtomicParsley.exe
  macOS:     AtomicParsley (chmod +x)

Download builds from the AtomicParsley project (e.g. wez/atomicparsley releases) or build from source.
Override path with environment variable POD240_ATOMICPARSLEY if needed.

Bundled binaries are not redistributed in this repository; copy them locally for development and packaging.
